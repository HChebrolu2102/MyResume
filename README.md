# Sri Harsha Chebrolu – Portfolio

Personal portfolio website built with **React + TypeScript + Vite**.

## Tech Stack
- React 18
- TypeScript
- Vite
- Framer Motion (animations)
- Lucide React (icons)
- CSS Modules

## Local Development

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

## Push to GitHub & Deploy to Vercel

```bash
# 1. Initialize git (if not already done)
git init

# 2. Add all files
git add .

# 3. Commit
git commit -m "Initial portfolio commit"

# 4. Add your GitHub repo as remote
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# 5. Push
git push -u origin main
```

Then go to **vercel.com**, import the repo, and click Deploy.
Vercel will auto-detect Vite and deploy with zero configuration.

## Customization
- Replace the `HC` avatar placeholder with your photo in `Portfolio.tsx`
- Update the resume download link in `Portfolio.tsx`
- Add your LinkedIn / GitHub URLs in the footer section
